MusicPlayer Application

Introduction

The MusicPlayer application demonstrates file management, audio output, and unit testing using Google Test and Google Mock. It showcases how various components, such as MusicPlayerController, Mp3FileManager, and SpeakerOutput, work together.

How to Build

  1. Clone the repository.
  2. Create a build directory: mkdir build && cd build.
  3. Run CMake: cmake ...
  4. Build the project: make.

How to Use

  • Run the main program: ./MusicPlayer
  • Run the unit tests: ./MusicPlayerTest
