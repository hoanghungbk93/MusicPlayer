\doxysubsection{Detailed Description}
A class for managing MP4 files. 

The \doxylink{class_mp4_file_manager}{Mp4\+File\+Manager} class provides methods to validate MP4 file paths and retrieve a list of supported video formats. It implements the \doxylink{class_file_manager_interface}{File\+Manager\+Interface}.
